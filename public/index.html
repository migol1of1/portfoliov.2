Portable Canine Scabies Detection Device - Prototype

An innovative Raspberry Pi based project that aims to assist veterinary professionals and dog owners in the early detection of canine scabies. The device utilizes a Raspberry Pi Camera Module to capture high-resolution images of a dog's affected area. The process of analyzing the images starts with training a pre-trained machine learning model to identify positive cases of canine scabies.

This research project that I was a part of combines my knowledge in machine learning, and embedded systems. Features include real-time image processing, AI-powered diagnosis, and a user-friendly web application for veterinary professionals and dog owners.

Technologies Used:

Numpy / Pillow Python MobileNetV2 TensorFlow Machine Learning
